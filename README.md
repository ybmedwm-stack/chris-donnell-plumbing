# Chris Donnell Plumbing & Heating — Website

A multi-page marketing site for **Chris Donnell Plumbing & Heating, Inc.**, a
family-owned plumbing and heating contractor based in York, Maine.

**Live preview:** https://ybmedwm-stack.github.io/chris-donnell-plumbing/

> This is a preview build for review. Content reflects publicly available
> business information and should be confirmed with the owner before launch.

---

## About the business

Chris Donnell Plumbing & Heating has served the Maine and New Hampshire Seacoast
since 2002. The shop is owner-operated and serves York County (ME) and the
surrounding Seacoast — from Wells and Ogunquit down through Kittery, Eliot, and
into Portsmouth and Dover in New Hampshire.

## Site structure

| `index.html`    | Home — hero, services overview, testimonials, service area   |

## Design

- **Aesthetic:** Premium contractor — deep navy + warm gold/copper accents
- **Typography:** Playfair Display (headings) + Inter (body), via Google Fonts
- **Layout:** CSS Grid + Flexbox, responsive at 880px and 520px breakpoints
- **Dependencies:** None. Pure HTML, CSS, vanilla JS. No build step.

## Deployment

Hosted on **GitHub Pages** from the `main` branch, root directory. Any commit
to `main` triggers an automatic rebuild within ~30 seconds.

To make changes:

1. Edit the file in this repo (GitHub web UI or local clone)
2. Commit to `main`
3. Refresh the live URL after a moment

## Browser support

Tested on current versions of Chrome, Safari, Firefox, and Edge — desktop and
mobile.
